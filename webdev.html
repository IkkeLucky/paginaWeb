Why Webdev

It was around 2019 that I started getting closer to code and what it has to offer. It began with some of my friends telling me to create something. To take something from other interests of mine like tennis or economy and to make a webpage or to develop an app for my services

I actually started discovering the incredible world of technology from building my own pc to creating functional websites. It was incredibly new for me and it was something that changes everyday. For me this is still a work in progress and I am still learning new things everyday.

For me the final decision of choosing this to be my probably last career path is that this is really the future. Programming is changing and will change the world in the next following years, it will be the next future universal language.

Certifications

Regarding certifications I have not yet received any since here in Argentina a career as Computer Science is not taught (or at least not the real one). So for the time being I am currently learning everything that I know via self taught programmes or courses and via freelancing.

I started learning the basic components and functionality of a computer until I knew enough to build my own. After that I started learning some basic math and logic especially to tackle functions with loops, booleans, algorithms and more. I had already studied two years in Biomedical Engineering so getting that knowledge back was not that hard.

Finally after knowing the basics of math and hardware components I started coding and learning basic html and css until I could end up learning more about the specifics of html and css and nowadays I started also learning about Javascript. I still need to develop my backend with Javascript and maybe add another language such as Typescript or Rust to expand my knowledge into backend development. This is what I aim for in the future
